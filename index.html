Caesar Cipher 🔐

Encode or decode your message instantly

📄 About This Project

I built a Caesar Cipher program using Python that can encode and decode messages by shifting letters based on a number given by the user. The program takes a message as input and moves each letter forward or backward in the alphabet depending on whether the user wants to encode or decode.

I made sure that spaces, symbols, and numbers stay the same so the message remains readable. The program runs in a loop, so the user can try multiple messages without restarting it.

Through this project, I learned how to use loops, conditions, functions, and string handling in Python, and also improved my problem-solving skills by handling different edge cases like symbols and large shift values.
